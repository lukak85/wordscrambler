Word Scrambler

About This Website

According to the research in Oxford University, humans are able to read sebtebces, as long as the first and the last letter of each word stays the same, while the inner part can be arranged any way you like and given the context, you'd still be able to read the word. This is because when we read a sentence, we do not read each word on it's own, but rather with the context given around it, which then our brain makes some sense of.

In my finding of that trait of the human mind, I decided to make a website which takes in a sentence, or really any sequence of words, seperated by spaces, and rearrages the inner part of each word, giving back the scrambled text which you would still be able to read, as per the experiment. Currently, the website only works when the words are only seperated by spaces.
